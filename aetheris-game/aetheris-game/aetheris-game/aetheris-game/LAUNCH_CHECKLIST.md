## Quick Deploy Commands

```bash
# 1. Setup audio directories
npm run setup:audio

# 2. Build and test locally
npm run build
npm run start

# 3. Deploy to preview
npm run deploy:preview

# 4. Deploy to production
npm run deploy
```
